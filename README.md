# Disease Trend Analysis Dashboard (Power BI)

## Project Overview

This project is a healthcare analytics dashboard built using Power BI to analyze disease trends, recovery rates, mortality rates, patient age distribution, treatment costs, and regional case distribution.

The dashboard provides meaningful insights into patient data and helps understand healthcare patterns for better decision-making.

---

## Objectives

* Analyze monthly disease trends
* Measure recovery and mortality rates
* Study disease distribution across patients
* Identify age group patterns
* Compare severity levels
* Analyze regional disease cases
* Evaluate average treatment costs

---

## Dataset

The dataset includes:

* Patient ID
* Admission Date
* Disease
* Age
* Gender
* Recovery Status
* Region
* Severity
* Treatment Cost

---

## Key KPIs

* Total Cases
* Recovery Rate
* Mortality Rate
* Average Age
* Average Treatment Cost

---

## Dashboard Visuals

* Monthly Disease Trend (Line Chart)
* Disease Distribution (Bar Chart)
* Age Group Analysis (Column Chart)
* Recovery Status (Pie Chart)
* Severity Analysis (Bar Chart)
